Pico to MCU Communication

Communication is via I2C using GPIO 6 & 7 on the Pico to PB8 & PB9 on the MCU. Pico is the I2C controller and MPU the I2C target, with a target address of 0x1F.

The Pico sends a one byte request or two byte command and receives a variable length response depending upon the request. The following table lists the possible requests and resulting responses.

LCD Display

From Code/picocalc_kbd_tester/lcdspi/lcdspi.h, the LCD controller is an ILI9488.

Compilation

In folder PicoBB/bin/pico, execute the command:

make BOARD=pico_w ADDON=PicoCalc STDIO=PicoCalc SOUND=SDL
